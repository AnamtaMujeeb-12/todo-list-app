# 📝 To-Do List Web App

A simple full-stack To-Do List application with user authentication and task management.

---

## 🚀 Features
- User Registration & Login
- Add new tasks
- Mark tasks as completed (update)
- Delete tasks
- User-specific todos

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Storage: JSON file (data.json)

---

## 📂 Project Structure
- index.html → Login/Signup page
- todo.html → Todo dashboard
- index.js → Backend server (APIs)
- data.json → Stores users & todos
- style.css → Styling

---

## ⚙️ How to Run
```bash
npm install
node index.js
